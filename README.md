## 🇺🇸 ENG

### Clone😎

```
git clone https://github.com/jovimoura/to-do-next
```

### Access the folder🤓

```
cd <folder>
```
### Install dependencies🤠
```
npm install
```
### Start application🤩
```
npm run start
```
### default port:🤗

```
http://localhost:3000/
```

<br>
